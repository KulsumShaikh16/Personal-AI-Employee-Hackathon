# Personal AI Employee - Silver Tier

## Features
- Gmail Watcher: New emails to Needs_Action .md files
- LinkedIn Real Post: Draft & real post using API
- File Watcher: Needs_Action monitor + plan generate
- Human-in-the-loop approval (Pending_Approval folder)
- Skills folder with reusable agent skills

## How to Run
1. pip install requests google-api-python-client etc.
2. credentials.json in Secrets/
3. python Scripts/gmail_watcher.py
4. python Scripts/linkedin_real_post.py
